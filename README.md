# Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn behavior using Python, SQL, and Power BI to identify key drivers and business insights.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI

## 📊 Key Insights
- Higher churn observed in monthly subscription plans
- Customers with complaints have significantly higher churn probability
- Early-stage customers (low tenure) are more likely to churn

## 📁 Project Structure
- notebook/ → Data analysis in Python
- sql/ → SQL queries for churn insights
- dashboard/ → Power BI dashboard
